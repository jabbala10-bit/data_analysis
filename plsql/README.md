# PL/SQL Analytics Library

This section expands the SQL library with Oracle PL/SQL examples tailored for a principal data analyst.

## Structure
- tutorials/ - progressive PL/SQL learning scripts from beginner to advanced topics
- scenarios/ - 50 scenario-based PL/SQL exercises for practical analytics work
- case_studies/ - five end-to-end business case studies covering the pipeline from data cleaning to report generation

## Learning Path
1. Start with the tutorial scripts to learn PL/SQL fundamentals.
2. Practice the scenario scripts to solve everyday analytic problems.
3. Use the case studies to build a portfolio-style workflow from raw data to reporting.

## Coverage Areas
- DDL, DML, and transaction control
- Cursors, collections, and bulk processing
- Procedures, functions, and packages
- Exception handling and logging
- Data cleaning, validation, and transformation
- Reporting and KPI generation
